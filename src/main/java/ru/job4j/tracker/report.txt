При удалении 100 заявок, CPU usage резко повысилась и упала на графике.
При добавлении 1000 заявок, Heap Memory Usage возрасло на графике.
При добавлении 100 000 заявок, Heap Memory Usage, как и CPU usage сильно возрасли на графиках.
При показе всех заявок, графики особо не изменились.
При добавлении 1 000 000 заявок, Heap Memory Usage упала с 250 мб до 50 мб, после чего возралса до прешних чисел снова,
такая тенденция сохранялась, пока программа не добавила все заявки, суммарно таких повторов было 8 раз. С CPU usage происодило тоже самое.
Анализ проводился инструментом jconsole.
